# VLC 播放

## 操作步骤

1. 新建名为 vlctest 的文件夹。
2. 右键文件夹打开终端，通过 wget 命令下载视频文件。
3. 打开 vlc。
4. 点击菜单栏中的 Tools 选项。
5. 点击 Preferences 选项。
6. 点击 Video 选项，在 Output 栏中选择 X11 video output（XCB)。
7. 点击右下角 Save。
8. 依次点击 Media -> Open File -> video1 -> open。

## 预期结果

视频播放成功，画面音频均有输出。

## 实际结果

视频播放成功，画面音频均有输出。但是画面卡顿，基本无法观看。
